Formulários em HTML

Os formulários são usados para coletar dados dos usuários. Eles são criados com a tag <form> e incluem vários tipos de campos de entrada. Abaixo, você encontrará um exemplo de um formulário simples.

Exemplo de Código HTML

<form action="destino_dos_dados">
    <label for="nome">Nome:</label>
    <input id="nome" type="text" placeholder="Digite seu nome" required>
    <br>

    <label for="sobrenome">Sobrenome:</label>
    <input id="sobrenome" type="text" placeholder="Sobrenome" required>
    <br>

    <label for="senha">Senha:</label>
    <input type="password" id="senha" placeholder="Digite a senha" required>
    <br>

    <label for="email">Email:</label>
    <input type="email" id="email" placeholder="Email" required>
    <br>

    <label for="data">Data:</label>
    <input type="date" id="data" required>
    <br>

    <label for="cor">Cor:</label>
    <input type="color" id="cor" required>
    <br>

    <label for="escolhas">Escolha:</label>
    <select id="escolhas" name="escolhas">
        <option selected disabled>Escolha</option>
        <option value="1">1</option>
        <option value="2">2</option>
        <option value="3">3</option>
    </select>
    <br>

    <label for="texto">Texto:</label>
    <textarea rows="10" cols="35" id="texto" placeholder="Digite seu texto aqui"></textarea>
    <br>

    <input type="submit" value="Enviar">
</form>

Explicação dos Campos

O formulário acima contém os seguintes campos:

  • Nome: Campo para inserir o nome do usuário.
  • Sobrenome: Campo para inserir o sobrenome do usuário.
  • Senha: Campo para inserir uma senha, que será ocultada.
  • Email: Campo para inserir um endereço de email.
  • Data: Campo para selecionar uma data.
  • Cor: Campo para escolher uma cor.
  • Escolha: Um menu suspenso para selecionar uma opção.
  • Texto: Área de texto para inserir informações adicionais.
