Aprendendo HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Abaixo estão algumas das tags HTML mais comuns, com exemplos e explicações sobre seus atributos.

<h1> a <h6>

Exemplo:

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>

Título Principal

Subtítulo

Seção

As tags de cabeçalho vão de <h1> (mais importante) a <h6> (menos importante). Você pode usar o atributo style para alterar a aparência, como style="color:red;".

<p>

Exemplo:

<p>Este é um parágrafo.</p>

Este é um parágrafo.

A tag <p> é usada para definir parágrafos. Você pode usar o atributo class para aplicar estilos CSS.

<a>

Exemplo:

<a href="https://www.exemplo.com">Visite o Exemplo</a>
Visite o Exemplo

A tag <a> é usada para criar links. O atributo href especifica o URL do link, e target="_blank" abre o link em uma nova aba.

<img>

Exemplo:

<img src="imagem.jpg" alt="Descrição da imagem">

A tag <img> é usada para inserir imagens. O atributo src especifica o caminho da imagem, e alt fornece uma descrição alternativa.

<ul> e <li>

Exemplo:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
  • Item 1
  • Item 2

A tag <ul> é usada para criar listas não ordenadas, enquanto <li> define cada item da lista. Você pode usar o atributo type na tag <ul> para alterar o estilo da lista.

<div>

Exemplo:

<div class="container">
    Conteúdo aqui
</div>
Conteúdo aqui

A tag <div> é um contêiner genérico para agrupar outros elementos. O atributo class pode ser usado para aplicar estilos CSS específicos.

<form>

Exemplo:

<form action="/submit" method="post">
    <input type="text" name="nome" placeholder="Seu nome">
    <input type="submit" value="Enviar">
</form>

A tag <form> é usada para criar formulários. O atributo action especifica para onde os dados do formulário serão enviados, e method define o método de envio (GET ou POST).

<table>

Exemplo:

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dados 1</td>
        <td>Dados 2</td>
    </tr>
</table>
Cabeçalho 1 Cabeçalho 2
Dados 1 Dados 2

A tag <table> é usada para criar tabelas. As tags <tr> definem linhas, <th> define cabeçalhos de coluna, e <td> define células de dados. Você pode usar o atributo border para adicionar bordas à tabela.

<span>

Exemplo:

<span style="color:blue;">Texto em azul</span>
Texto em azul

A tag <span> é usada para aplicar estilos a partes de um texto. O atributo style pode ser usado para definir a cor, tamanho e outros estilos.

<strong> e <em>

Exemplo:

<strong>Texto em negrito</strong>
<em>Texto em itálico</em>
Texto em negrito Texto em itálico

A tag <strong> é usada para dar ênfase ao texto, geralmente exibido em negrito, enquanto <em> é usada para ênfase em itálico. Ambos podem ser estilizados com CSS.

Esta página é um exemplo básico de HTML. Explore e modifique os exemplos para aprender mais sobre HTML e suas funcionalidades. Você pode adicionar mais tags e atributos conforme necessário para expandir seu conhecimento e prática.
