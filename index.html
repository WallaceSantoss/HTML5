HTML = Hyper Text Markup Language

Formato básico de web para estrutura: .html, .htm

Estrutura Básica

Todo documento HTML começa com a declaração do tipo de documento e é composto por elementos estruturais:

<!DOCTYPE html>
<html>
<head>
    <title>page title</title>
</head>
<body>
    <h2>Heading content</h2>
    <p>Paragraph</p>
</body>
</html>

Hipertextos

Hipertextos são links que conectam um documento HTML a outro. Eles são criados usando a tag <a>:

<a href='caminho' target='_blank'>Link</a>

Linguagem de Marcação

HTML usa tags para marcar os elementos de um documento. Algumas tags comuns incluem:

<title></title>

Tags Comuns:

  • <img> - Imagem
  • <video> - Vídeo
  • <audio> - Áudio

Lista Desordenada:

Uma lista desordenada é criada usando as tags <ul> e <li>. Exemplo:

<ul>
    <li>Arroz</li>
    <li>Feijão</li>
    <li>Macarrão</li>
</ul>

Exemplo exibido:

  • Arroz
  • Feijão
  • Macarrão

Meta Tags:

Meta tags fornecem metadados sobre o documento HTML e são usadas principalmente pelo navegador e motores de busca:

  • <meta charset="UTF-8"> - Define o conjunto de caracteres como UTF-8
  • <meta http-equiv="X-UA-Compatible" content="IE=edge"> - Compatibilidade com Internet Explorer
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> - Adaptabilidade para diferentes tamanhos de tela
  • <meta name="description" content="site"> - Descrição do site
  • <meta name="keywords" content="site, html"> - Palavras-chave
  • <meta name="author" content="wallace"> - Autor do site

Títulos:

Títulos são usados para estruturar o conteúdo em diferentes níveis de importância:

  • <h1> - Título principal (maior)
  • <h2> - Subtítulo
  • <h3> ...
  • <h4> ...
  • <h5> ...
  • <h6> - Subtítulo menor (menor)

Texto:

Tags para formatação de texto:

  • <p> - Parágrafo
  • <b> - Negrito
  • <strong> - Dá ênfase ao texto
  • <hr> - Linha horizontal
  • <br> - Quebra de linha
  • <i> - Itálico
  • <u> - Sublinhado
  • <small> - Diminui o texto
  • <del> - Texto excluído
  • <sup> - Texto superior
  • <sub> - Texto inferior
  • <mark> - Marcação de texto

Citações:

Tags para citações e referências:

  • <abbr title="significado"> - Abreviação
  • <address> - Endereço
  • <cite> - Citação de obras
  • <q> - Citação curta
  • <blockquote> - Citação longa, indentada
  • <bdo dir="rtl"> - Inverte a direção do texto

Comentários:

Os comentários são usados para deixar notas no código que não serão exibidas no navegador:

<!-- comentário -->

Links:

Os links são criados com a tag <a> e podem abrir em uma nova aba usando o atributo target:

<a href='caminho' target='_blank'>Link</a>
