Tabelas:

As tabelas são usadas para organizar dados em formato tabular. Elas são criadas com as tags <table>, <tr>, <th>, e <td>:

Exemplo de Tabela Simples:

<table border="1">
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>Fulano</td>
        <td>30</td>
    </tr>
    <tr>
        <td>Ciclano</td>
        <td>25</td>
    </tr>
</table>

Exemplo exibido:

Nome Idade
Fulano 30
Ciclano 25

Exemplo de Tabela Complexa:

<table border="1">
    <tr>
        <th>Pele</th>
        <th>Olhos</th>
        <th>Altura</th>
    </tr>
    <tr>
        <td>Branca</td>
        <td>Castanhos</td>
        <td>1,70m</td>
    </tr>
    <tr>
        <td>Negra</td>
        <td>Verdes</td>
        <td>1,80m</td>
    </tr>
</table>

Exemplo exibido:

Pele Olhos Altura
Branca Castanhos 1,70m
Negra Verdes 1,80m

Explicação das Tags:

  • <table>: Define a tabela.
  • <tr>: Define uma linha na tabela.
  • <th>: Define uma célula de cabeçalho, que é geralmente exibida em negrito e centralizada.
  • <td>: Define uma célula de dados na tabela.
