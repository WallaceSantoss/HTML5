Listas Não Ordenadas:

Listas não ordenadas são criadas com as tags <ul> e <li>:

<ul>
    <li>Arroz</li>
    <li>Feijão</li>
    <li>Batata</li>
    <li>Alho</li>
    <li>Bombril</li>
</ul>

Exemplo exibido:

  • Arroz
  • Feijão
  • Batata
  • Alho
  • Bombril

Listas Ordenadas:

Listas ordenadas são criadas com as tags <ol> e <li>:

<ol>
    <li>Arroz</li>
    <li>Carne</li>
</ol>

Exemplo exibido:

  1. Arroz
  2. Carne

Explicação das Tags:

  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define um item de lista, que pode ser usado em ambas as listas.
